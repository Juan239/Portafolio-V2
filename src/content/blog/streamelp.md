Ha pasado tiempo de que subí este proyecto, sin embargo nunca lo había agregado al blog, Streamelp es un sistema que nace de la idea de que los streamers puedan tener más interacción con sus espectadores, de momento el sistema cuenta con la posibilidadad de crear "consultorios" los cuales los espectadores pueden responder dependiendo de la temática dada por el streamer.

Streamelp está desarrollador con NodeJS y Express, usando TypeScript, cuenta con una base de datos PostgreSQL, ocupando PrismaORM para las consultas. También cuenta con integración con la API de Kick, para poder autenticarse usando la cuenta personal de cada usuario.

De momento el desarrollo de este sistema se encuentra en pausa debido al poco tiempo que tengo para desarrollar proyectos personales, aunque en caso de bugs o problemas de seguridad puedo hacerme un tiempo para actualizarlo.
