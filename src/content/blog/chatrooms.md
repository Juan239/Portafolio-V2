Pasaron más de 3 meses desde el último post, hoy vengo a hablar de un nuevo proyecto que hice, es algo simple, ya que solo lo hice para probar cosas de la VPS que compré, es un chat en tiempo real en la que un usuario puede crear una sala y puede compartir el enlace para que otras personas puedan unirse, el backend está desarrollado con NodeJS, Express y Typescript, Socket.IO para la gestión de websockets. Para el frontend solo use React y Tailwind.

Todos los mensajes son guardados en una base de datos, por si acaso.
